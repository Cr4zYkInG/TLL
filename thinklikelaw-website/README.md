# ThinkLikeLaw LawSovereign

A web-based platform for UK law students, featuring module tracking, AI-powered study tools, and a comprehensive dashboard.

## Local Development

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Local Server**:
    ```bash
    npm start
    ```
    Access the site at `https://www.thinklikelaw.com`.

3.  **Cloudflare Worker Development**:
    To run the email worker locally:
    ```bash
    npm run dev:worker
    ```

## Deployment

deploy the worker:
```bash
npm run deploy:worker
```
